LaneRouter

A reference implementation of the Two-Lane Rule: sensitive data takes the local lane, always. Every request is classified, routed, and audit-logged. The log stores metadata only — never the text.

Demo simulation — no live backend on this page. Clone the repo and run python3 server.py for the real router with your own local models. Sample data below is synthetic.
The test: would this data need a BAA? Then it never leaves hardware you control. Detectors are conservative on purpose: suspicion routes local.
